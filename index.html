Hardware & Software

Every device is made of two inseparable parts: physical components you can touch, and invisible programs that make them useful.

🔩 Hardware

The physical parts of a computer — things you can see and touch. Examples: your screen, keyboard, hard drive, and the chips inside your phone.

💿 Software

Instructions that tell hardware what to do. Software has no physical form — it's code stored and run on your device.

🗂️ Types of Software

Software comes in layers, each with a different job:

🖥️ Operating System — Windows, macOS, Android 📱 Applications — Word, Chrome, Instagram ⚙️ Drivers — Make devices talk to the OS 🔧 Utilities — Antivirus, disk cleanup tools

📖 Key Terms

CPU (Central Processing Unit)
The "brain" — executes instructions and processes data.
RAM (Random Access Memory)
Short-term memory your computer uses while actively working. Clears when powered off.
Storage (HDD / SSD)
Long-term memory that holds your files even when powered off.
Operating System (OS)
Software that manages all hardware and lets other programs run (e.g., Windows, macOS).

Networks & the Internet

A network is any group of connected devices. The Internet is the world's largest network — billions of devices all linked together.

📶 Types of Networks

  • LAN — Local Area Network (home, office)
  • WAN — Wide Area Network (cities, countries)
  • Internet — Global WAN connecting billions
  • Wi-Fi — Wireless LAN using radio waves

🔀 How Data Travels

Data is broken into small packets, each labeled with source/destination IP addresses. A router directs each packet — like a post office for data.

🌍 Internet vs. the Web

People often use these interchangeably — but they're different!

🌐 The Internet
The global infrastructure — cables, routers, and protocols that connect devices worldwide.
🕸️ The Web (WWW)
A service that runs on the internet — websites and web apps accessed through a browser.

🎯 Match the Terms

Match each networking term to its correct definition.

📖 Key Terms

IP Address
A unique number assigned to every device on a network — like a home address for data.
Router
Device that directs data packets between networks and devices.
Protocol
A standard set of rules devices use to communicate (e.g., HTTP, TCP/IP).
Bandwidth
The maximum amount of data that can travel through a connection per second.

Cybersecurity & Online Safety

Cybersecurity means protecting computers, networks, and data from unauthorized access. It's everyone's responsibility — not just IT professionals.

🦠 Malware

Malicious software designed to harm. Includes viruses, ransomware, and spyware.

🎣 Phishing

Fake emails or sites that trick you into giving up passwords or financial info.

🔑 Weak Passwords

Simple passwords are cracked in seconds. Reusing passwords multiplies the risk.

🛡️ How to Stay Safe Online

✅ Long, unique passwords — or a password manager
✅ Two-factor authentication (2FA) wherever possible
✅ Keep software updated — updates patch security holes
✅ Verify before you click — check sender addresses
✅ Back up your data — 3-2-1 rule (3 copies, 2 types, 1 off-site)
✅ Use HTTPS sites — look for the 🔒 in your browser

Cloud Computing

Instead of running everything locally, cloud computing lets you store data and run programs on powerful remote servers — accessed over the internet.

☁️ What Is the Cloud?

When you save a photo to Google Photos, stream Netflix, or edit a Google Doc — you're using the cloud. Your data lives on servers in large data centers around the world.

💡 Analogy: Think of the cloud like electricity. You don't generate your own power — you plug in and use what you need. Cloud computing works the same way.
🗄️
IaaS
Infrastructure as a Service
Raw servers & storage
e.g. AWS, Azure
🏗️
PaaS
Platform as a Service
Dev tools & runtime
e.g. Heroku, Firebase
📦
SaaS
Software as a Service
Ready-to-use apps
e.g. Gmail, Zoom

✅ Benefits of Cloud

  • Access files from any device, anywhere
  • Automatic backups — less data loss risk
  • Scale up or down as needed
  • Lower upfront cost (no expensive servers)
  • Real-time collaboration

⚠️ Considerations

  • Requires an internet connection
  • Privacy — who can see your data?
  • Subscription costs add up
  • Vendor lock-in (hard to switch)
  • Outages affect everyone at once
